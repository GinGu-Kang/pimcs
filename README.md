# pimcs
### 📖 내용

- 실시간으로 무게측정 IOT 기기를 통해 재고 수량 파악
- 입출고 내역을 기록하여 그래프 시각화
- 재고목록과 입출고 기록 CSV 리포트 출력
- 최소 잔여수량 미달시 자동발주 실행
- 시연 영상: [https://www.youtube.com/watch?v=Y6_GGQf_fus](https://www.youtube.com/watch?v=Y6_GGQf_fus)
